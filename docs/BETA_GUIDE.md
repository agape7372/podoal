# podoal 친구 베타 가이드

> 목적: 사용자 + 친구 몇 명이 각자 폰에 설치하고 **서로 응원·포도판 선물·깜짝 선물·릴레이를 실제로 주고받는** 단계. 아래 §초대문은 친구에게 그대로 복붙해서 보낼 수 있다.
> 배포: `https://podoal-rouge.vercel.app` (main push 시 Vercel 자동 배포).

## 시작 전 — 운영자(사용자) 체크리스트

| 항목 | 상태(2026-07-05) | 조치 |
|------|------------------|------|
| **가입 경로** | OAuth 3사 전부 **게스트(체험) 모드**(`real:false`) — 매번 임시 신원이라 지속 친구관계에 부적합 | 친구들은 **이메일 가입**으로 안내(아래 초대문). 진짜 소셜 로그인 원하면 Vercel에 `GOOGLE/KAKAO/NAVER_CLIENT_ID/SECRET` 설정(docs/OAUTH_SETUP.md) |
| **웹푸시(알림)** | VAPID env 설정 시에만 동작 | 주고받기 체감의 핵심 → **베타 전 Vercel에 `NEXT_PUBLIC_VAPID_PUBLIC_KEY`·`VAPID_PRIVATE_KEY`·`VAPID_SUBJECT` 설정 권장**(docs/PUSH_SETUP.md). 미설정이어도 앱 안에서는 실시간(SSE) 수신됨 — 앱을 닫아둔 사이 알림만 안 옴 |
| **cron 알림** | GitHub Actions 3종 + `CRON_SECRET` | 리마인더·데일리 넛지 쓰려면 Vercel·GitHub 양쪽 `CRON_SECRET` 일치 확인 |
| **친구 찾기** | 이름/이메일 검색(`/friends`) | 친구끼리 **가입 시 쓴 이메일 또는 표시 이름**을 공유해두면 서로 검색 가능 |

## 친구에게 보낼 초대문 (복붙용)

> 🍇 **포도알(podoal) 같이 해볼래?**
> 습관을 포도알 하나씩 채우는 앱이야. 서로 응원도 보내고 포도판을 선물할 수도 있어.
>
> **1. 설치** (앱스토어 말고 웹앱이야)
> - 링크 열기: https://podoal-rouge.vercel.app
> - **아이폰(사파리)**: 공유 버튼 → "홈 화면에 추가"
> - **안드로이드(크롬)**: 메뉴(⋮) → "홈 화면에 추가" / "앱 설치"
>
> **2. 가입**: 첫 화면에서 **이메일로 가입**(이름·이메일·비밀번호 8자↑). ※ 소셜 로그인 버튼은 지금 체험용이라 이메일로 해줘.
>
> **3. 친구 추가**: 하단 "친구" 탭 → 검색에 내 이름(또는 이메일 `<여기에 본인 이메일>`) 넣고 친구 요청. 내가 수락할게.
>
> **4. 같이 해보기**: 아래 시나리오대로!

## 같이 해볼 시나리오 (베타 테스트 항목)

1. **서로 응원 보내기**: 친구 탭 → 친구 → "응원 보내기" → 이모지+문구 선택. 상대 알림함(💌 소통)에 도착하는지.
2. **포도판 선물**: 내 포도판 상세 → 메뉴 → "포도판 선물하기" → 친구 선택 + 메시지. 친구가 받아서 여는지. (같은 판을 두 번 보내면 "이미 선물했어요" 뜨는 게 정상 — 중복 방지.)
3. **깜짝 선물 심기**: 친구 포도판 상세 → 아직 안 채운 칸에 깜짝 선물 심기(친구가 "깜짝 선물 받기"를 켜둔 판만). 친구가 그 칸을 채우면 발견되는지.
4. **릴레이(포도동) 한 판**: 친구 탭 → 릴레이 → 만들기 → 친구 초대. 순차/그룹 모드로 바통 넘기거나 같이 완주.
5. **알림 켜기**: 설정 → 알림 → 마스터 토글 + 알림 권한 허용. 앱 닫아둔 상태에서 응원/선물 받을 때 푸시 오는지(VAPID 설정 시).
6. **포도판 채우고 완성**: 알 채우는 촉감, 중간 보상, 완성 축하(액체 차오름), 와이너리 병 입고.

## 알려진 제약 (베타)

- 소셜 로그인은 아직 체험 모드 → 이메일 가입 사용.
- VAPID 미설정 시 앱을 닫아둔 사이 푸시 알림이 안 옴(앱 열려 있으면 실시간 수신은 됨).
- 데이터는 실제 저장됨(장난 계정도 남음) — 친구끼리만 공유 권장.
