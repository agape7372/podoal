**포도알 개인정보처리방침**

포도알(이하 "서비스")은 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 등 관련 법령을 준수합니다.

## 1. 수집하는 개인정보 항목

### 필수 (회원가입·서비스 제공)
| 항목 | 수집 시점 | 목적 |
|------|----------|------|
| 이메일 주소 | 이메일 가입 시 | 계정 식별·로그인 |
| 비밀번호(암호화 저장) | 이메일 가입 시 | 계정 보안 |
| 닉네임·프로필 아바타 | 가입·프로필 수정 시 | 서비스 내 표시 |
| 소셜 로그인 식별자 | 구글·카카오·네이버 로그인 시 | 계정 연결(비밀번호 미수집) |

### 서비스 이용 과정에서 생성
| 항목 | 목적 |
|------|------|
| 습관 기록(포도판·포도알·보상·메시지·타임캡슐 등) | 서비스 핵심 기능 제공 |
| 업로드 이미지(보상 사진·커스텀 알 사진) | 이용자가 직접 등록한 콘텐츠 표시 |
| 웹 푸시 구독 정보(브라우저 엔드포인트) | 이용자가 켠 알림 발송 |

### 선택 (동의 시에만 — 익명 사용 통계)
| 항목 | 목적 |
|------|------|
| 익명화된 서비스 이용 이벤트(화면 이동·기능 사용 여부, 내부 식별자 기준) | 서비스 개선·기능 우선순위 판단 |

- 이용 통계는 **최초 실행 시 동의한 경우에만** 수집합니다. 거절하거나 응답하지 않으면 아무것도 수집하지 않습니다.
- 통계에는 이메일·닉네임·메시지 내용 등 **개인을 알아볼 수 있는 정보를 포함하지 않습니다**.

## 2. 개인정보의 처리 목적

1. 회원 관리: 가입·로그인·계정 식별·탈퇴 처리
2. 서비스 제공: 습관 기록·친구·응원·선물·알림 등 핵심 기능
3. 서비스 개선(동의 시): 익명 통계 기반 기능 개선

## 3. 보유 및 이용 기간

- 회원 정보·습관 기록: **회원 탈퇴 시 즉시 삭제** (탈퇴 시 계정과 연결된 데이터가 함께 삭제됩니다)
- 익명 이용 통계: 수집일로부터 **최대 24개월** 보관 후 파기
- 법령상 보존 의무가 있는 경우 해당 기간 동안 보존
