## 현황 스냅샷 (2026-07-05)

습관 추적 PWA, 클레이모피즘. 코어 완성: 포도판 생성·채우기(낙관+Serializable)·보상·중간보상·캡슐·와이너리 7티어·친구·응원·선물·깜짝선물·릴레이(순차/그룹)·통계·바인·PWA·웹푸시·cron 3종. 전면 감사 결과 Critical/High 0, Med 3·Low 3 전건 수정 완료(감사 보고서 참조). 아키텍처 건실, 기술부채 낮음.

## P1 — 안정화 잔여 (다음 우선)

- **목표**: 감사 제안(§5) 중 채택분 + 친구 베타 피드백 반영.
- **후보**: rewards 카드 → 원본 보드 링크(2차 어포던스, UX 검토 후), 친구 찾기 UX 다듬기(베타에서 걸리면), 실기기 드래그 정렬 회귀 재확인(#94~#103).
- **선행조건**: 친구 베타(`docs/BETA_GUIDE.md`) 1라운드 완료.
- **완료 정의**: 베타 발견 이슈가 감사 보고서에 등재·처리, 코어 저니 실기기 무결.

## P2 — 네이티브 홈화면 위젯

- **목표**: iOS/Android 홈화면 위젯(오늘 진행률 + 딥링크·원탭 채우기).
- **상세**: `docs/WIDGETS_NATIVE_PLAN.md`(Capacitor 셸 + WidgetKit/Glance + per-user 위젯 토큰). 웹 그라운드워크(`/api/widgets/today`·manifest shortcuts·daily-nudge)는 구현됨.
- **선행조건**: Apple Developer($99/년)·Google Play($25) 계정, 출시 전 비공개 배포 게이트 결정.
- **완료 정의**: 내부 테스트(TestFlight/Play Internal) 트랙에 위젯 동작.

## P3 — 수익화 (포도알 꾸미기)

- **목표**: 스킨(품종) 상점 — 진행은 무료, 코스메틱만 유료. 광고 보상 무과금 경로.
- **상세**: `docs/MONETIZATION_PLAN.md`. 단계 R1 스킨 무료 출시 → R2 지갑+토스페이먼츠 → R3 광고 보상(네이티브 셸).
- **선행조건**: R2는 PG 계약·사업자 등록·전자상거래 표시. R3는 P2 네이티브 셸 + AdMob SSV.
- **완료 정의(R1)**: `GrapeSkin`·`Board.skinId` 스키마 + 꾸미기 UI + 무료 스킨 배포, 리텐션 반응 측정.
- **⚠ 착수 전 필독**: MONETIZATION_PLAN §4(IAP 정책 결정 트리)·§5(가드레일), PRINCIPLES §3(데이터 레이어 게이트)·수익화 가드레일.

## P4 — 성장·완성도 후보 (아이디어 주차장 — 착수 조건 명시)

- **stats 히트맵 드릴다운**: 셀 탭 → 날짜별 활동 상세. 조건: 신규 화면 UX 설계.
- **relay 완주 기념물**: 릴레이 완주 자체의 기록물(현재는 병만). 조건: 세계관 확장 결정.
- **vine 페이지네이션**: 현재 전체 로드. 조건: 헤비유저 데이터량이 성능 문제로 드러날 때(현재는 인덱스로 커버).
- **`transition-all` 레거시 치환**: 신규 코드는 REVIEW_CHECKLIST로 차단. 기존 ~20곳은 모션 회귀 위험 대비 이득 낮아 기회 있을 때 점진 치환(일괄 개조 금지).
- **messages 발신자 → 프로필 링크**: 친구 게이트(PA-006) 적용으로 발신자는 친구뿐 → 안전. 조건: UX 우선순위.

## Non-goals (명시적으로 안 하는 것)

- 진행(알 채우기·보상·릴레이)의 유료화·광고 게이팅 — 영원히 무료(PRINCIPLES §1·수익화 가드레일).
- 가챠·랜덤박스·만료 압박.
- 카피·티어명·템플릿의 임의 변경(디자인이 변화를 나른다 — 선물·공유·메시지 문구만 승인된 예외).
- 강제 전면광고·배너.
