## P3 — 수익화 (포도알 꾸미기)

- **목표**: 스킨(품종) 상점 — 진행은 무료, 코스메틱만 유료. 광고 보상 무과금 경로.
- **상세**: `docs/MONETIZATION_PLAN.md`. 단계 R1 스킨 무료 출시 → R2 지갑+토스페이먼츠 → **R2.5 구독 "포도클럽"(코스메틱 패스 — §8, R2 가동 후 검토)** → R3 광고 보상(네이티브 셸).
- **선행조건**: R2는 PG 계약·사업자 등록·전자상거래 표시. R3는 P2 네이티브 셸 + AdMob SSV.
- **완료 정의(R1)**: `GrapeSkin`·`Board.skinId` 스키마 + 꾸미기 UI + 무료 스킨 배포, 리텐션 반응 측정.
- **⚠ 착수 전 필독**: MONETIZATION_PLAN §4(IAP 정책 결정 트리)·§5(가드레일), PRINCIPLES §3(데이터 레이어 게이트)·수익화 가드레일.

## P4 — 성장·완성도 후보 (아이디어 주차장 — 착수 조건 명시)

- **stats 히트맵 드릴다운**: 셀 탭 → 날짜별 활동 상세. 조건: 신규 화면 UX 설계.
- **relay 완주 기념물**: 릴레이 완주 자체의 기록물(현재는 병만). 조건: 세계관 확장 결정.
- **vine 페이지네이션**: 현재 전체 로드. 조건: 헤비유저 데이터량이 성능 문제로 드러날 때(현재는 인덱스로 커버).
- **`transition-all` 레거시 치환**: 신규 코드는 REVIEW_CHECKLIST로 차단. 기존 ~20곳은 모션 회귀 위험 대비 이득 낮아 기회 있을 때 점진 치환(일괄 개조 금지).
- **messages 발신자 → 프로필 링크**: 친구 게이트(PA-006) 적용으로 발신자는 친구뿐 → 안전. 조건: UX 우선순위.
