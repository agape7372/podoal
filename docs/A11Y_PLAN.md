# podoal 접근성 팩 설계 (A11Y_PLAN)

> 상태: **설계(부분 구현)** — PRODUCT_PLAN §8 제안-3 채택분(2026-07-08)의 실행 정본. ROADMAP 백로그 "접근성 팩" 행이 이 문서를 가리킨다.
> 근거: REQ-08 글자 크기(3명 — P11·P12·P13 중장년 전원) · REQ-09 색각 이중 부호화(P15) · ABS-14 소셜 숨김(P21 심리 안전).
> 기존 자산(재추가 금지): 전역 `:focus-visible` 링 · 전역 `prefers-reduced-motion` 백스톱 · AA 텍스트 규약(`text-warm-sub`) · `tabular-nums` · 16px 입력 · 핀치줌 허용 — PRINCIPLES §5.

## 1. 소셜 숨김 (ABS-14) — ✅ 이번 마라톤 구현

- 홈 "친구 소식" 피드 토글(`설정 → 표시`), 기본 표시. 카드: `docs/cards/2026-07-08-W3-social-hide.md`.
- 설계 원칙: 숨김은 **감상 계층만** 끈다 — 친구·응원·선물 기능 자체는 불변(관계 해자 유지). 추후 확장 후보: 완성 축하 팝업의 타인 진행 노출 검토(현재 없음 — 확장 불요 판단).

## 2. 글자 크기 (REQ-08) — 설계, 실기기 QA 선행 필요

**방식(권고)**: 설정 3단(기본 100% / 크게 112.5% / 아주 크게 125%) → `<html data-font-scale>` 속성 + CSS `font-size` 오버라이드. 전 타이포가 rem 기반(tailwind text-*)이라 루트 스케일만으로 전체 반영.

- 저장: `AppSettings.fontScale?: 'base' | 'lg' | 'xl'`(additive) — dual-store 아님(Zustand만).
- 적용 지점: `(app)/layout.tsx`에서 html 속성 세팅(웰컴 페이지 포함 여부는 QA 후 결정 — 설정 접근이 로그인 후라 v1은 (app)만).
- **리스크(QA 게이트)**: ①실측 px 튜닝 지점 — InstallPrompt 제목 13/14px 실측(컴포넌트 주석), 홈 카드 truncate, 레이아웃 불변식 3(pb-[160px]) ②GrapeBoard는 px 기하(grapeSize)라 무영향(의도 — 판은 스케일 제외) ③125%에서 4탭 네비 라벨 줄바꿈.
- 카드 분할: A11Y-1(설정 UI+속성 배선, sonnet) → A11Y-2(125% 전 화면 스윕·보정, sonnet+실기기 확인은 사용자).
- **착수 조건**: 실기기 확인 가능 시점(P1 실기기 세션과 합류 권장).

## 3. 색각 이중 부호화 (REQ-09) — ✅ 히트맵 완료(2026-07-11)

현황 감사: 상태 전달이 색 단독인 곳 — ①stats 히트맵(초록 5단계 채도만) ②채움 진행바(보라 농도) ③grape-ripening 청→보라(신규). 그 외 배지·칩은 아이콘/텍스트 동반이라 합격.

- **히트맵(주 대상)**: 셀 내 도트 크기 병행(활동량 비례 반지름) 또는 셀 탭 시 숫자 노출 — 권고: **도트 크기 병행**(항상 보임, 팔레트 토글보다 유지비 낮음). P4 "히트맵 드릴다운"과 같은 파일이라 동시 착수 권장.
- **진행바**: 퍼센트 숫자 이미 병기 — 합격, 무작업.
- **grape-ripening**: 색+텐션 펄스에 더해 "익는 중" 보조 라벨이 카드(W3-cadence-ripen)에 포함돼 이중 부호화 충족 — 무추가.
- ✅ 히트맵 도트 병행 완료(2026-07-11, 카드 `2026-07-11-W1B` — 색 버킷 동일 5단계 도트, WCAG 1.4.11 대비 계산, 그레이스케일 판독 검증). P4 드릴다운과는 비병합(착수 조건 미충족 — 도트는 후속을 막지 않음).
